Cocos2d-x的菜单

发表于 2016-08-30   |   分类于 Cocos2d-x基础知识系列   |   | 阅读量 次

　　一款游戏常常会有一个菜单的，玩家可通过菜单设置游戏的一些属性，例如游戏声音是否开启，音量大小等等。那么Cocos2d-x中是怎样实现菜单的呢？下面船长就来说说Cocos2d-x中的菜单以及菜单项，大家请往下看。

菜单类（Menu）

　　Cocos2d-x中的菜单使用Menu类来实现，该类是一个容器，用来存放各种菜单项。Menu类继承关系图如下：


　　通过继承关系图砸门可以看出Menu的父类是Layer，所以它是一个容器，但是通常只用来装载各种菜单项而已。[/手动尴尬]（貌似说了两遍Menu是一个容器通常只用来装载菜单项。）
　　Menu类的常用函数如下：
　　♠ static Menu＊ create()：创建一个空菜单。
　　♠ static Menu＊ create(MenuItem＊ item, …)：根据菜单项对象创建一个菜单，菜单项为一个或多个。
　　♠ static Menu＊ createWithArray(const Vector＜MenuItem＊＞& arrayOfItems)：通过给定的一组菜单项对象创建菜单。
　　♠ static Menu＊ createWithItem(MenuItem＊ item)：通过给定的一个菜单项对象创建菜单，之后可以通过addChild函数添加其他菜单项。
　　♠ void alignItemsVertically()：垂直对齐菜单项。
　　♠ void alignItemsVerticallyWithPadding(float padding)：以指定间距(padding)垂直排列菜单项。
　　♠ void alignItemsHorizontally()：水平对齐菜单项。
　　♠ void alignItemsHorizontallyWithPadding(float padding)：以指定间距(padding)水平排列菜单项。

菜单项（MenuItem）

　　Cocos2d-x中提供了多种菜单项，用于展现不同风格的游戏界面。MenuItem和它的子类可以加到Menu中形成菜单，MenuItem的父类及其子类图解如下：


　　MenuItem类是所有菜单项的基类，so砸门不要直接使用该类，因为该类中并木有具体显示的功能，使其作为所有菜单项的父类，主要提供以下两个功能：
　　♥ 提供基本菜单项的状态：正常、选中和无效。
　　♥ 为菜单项对象实现了基本的回调函数机制。当游戏玩家单击按下一个菜单项时，通常需要执行一个动作，比如进入游戏场景或打开菜单选项等，此时就会调用一个函数来进行相关处理，这个函数就是菜单项的回调函数，砸门只需要在回调函数中实现相关功能就ok啦。

标签菜单项（MenuItemLabel）

　　MenuItemLabel类是使用文字定义的菜单项，包括MenuItemAtlasFont和MenuItemFont两种定义菜单项字体的方式，之前博客中说到的LabelTTF、LabelBMFont、LabelAtlas和Label标签对象都可以放置在该菜单项中。该菜单项常用的create函数如下：
　　♣ static MenuItemLabel＊ create(Node＊ label, const ccMenuCallback& callback)
　　以上create函数的第一个参数是菜单项中显示的标签对象，第二个参数是玩家单击时调用的回调函数。

自定义字体菜单项（MenuItemAtlasFont）

　　该类继承自MenuItemLabel类，它支持使用LabelAtlas来创建菜单项的对象，该类常用的create函数如下：
　　♦ static MenuItemAtlasFont＊ create(const std::string& value, const std::string& charMapFile, int itemWidth, int itemHeight, char startCharMap, const ccMenuCallback& callback)
　　以上create函数通过一个LabelAtlas对象来创建菜单项。第一个参数是菜单项中显示的文字内容，第二个参数是字体图集的名字，第三个参数是字体的宽度，第四个参数是字体的高度，第五个参数是字体图集开始的字符，第六个参数是单击时的回调函数。

字体菜单项（MenuItemFont）

　　该类同样继承自MenuItemLabel类，它支持直接使用文字来创建菜单项，文字字体是系统自带的。该类常用的create函数如下：
　　♠ static MenuItemFont＊ create(const std::string& value, const ccMenuCallback& callback)
　　以上create函数直接使用文字内容来创建菜单项。第一个参数是菜单项中显示的文字内容，第二个参数是单击时的回调函数。可以通过setFontName函数设置系统所支持的字体，通过setFontSize设置字体大小。

精灵菜单项（MenuItemSprite）

第一个Cocos2d-x项目之HelloWorld详解

发表于 2016-07-24   |   分类于 Cocos2d-x基础知识系列   |   | 阅读量 次

　　前几篇博文主要概述了Cocos2d-x中需要知道的基本概念及其使用方法等等，船长将通过这篇博文与大家一起创建第一个Cocos2d-x项目，当然这个项目毫无疑问就是HelloWorld。船长不仅仅只是创建第一个Cocos2d-x的项目，而且还会简单的分析一下项目结构啥的。下面就让我们开始创建第一个项目吧！

创建HelloWorld项目

　　在创建HelloWorld项目之前，你一定要确保已经配置好了环境变量，至于环境变量怎么配置请自行百度或Google，如果还是不明白可以评论问船长，也可以在留言板中留言。

命令行创建

　　在命令窗口中输入命令行：
　　cocos new Name -p PackName -l LanguageName -d D:\cocos2d-x\project
　　后回车就ok了，命令窗口（Dos窗口）进入方法：按Windows+R组合键输入cmd回车等。
　　命令行中Name为新建项目的名字，PackName为包名，LanguageName为新建项目的编程语言，-d后面为新建项目放置的位置。注意：包名为三级的，例如as.fad.fdew等，也就是说要有三个字段两个点。每一款游戏或应用都是通过包名来区分的，例如你的手机里已经安装了一款包名为com.youxi.www的游戏或应用A，你又下载了另外一款游戏或应用B且包名也为com.youxi.www，当你安装B时就会替换掉A。LanguageName之前是-l（小写的L），因为目前Cocos2d-x引擎支持新建C++和Lua的项目，所以这里的语言可以为cpp或lua。

项目结构分析

　　创建了一个项目之后，我们要知道这个项目的文件结构是啥样的，每个文件是干啥的，以及该项目的工作流程代码控制是怎样的，下面就由船长给大家简单地讲讲。

项目的文件结构

　　打开你新建的项目文件夹，你可以看到如下图所示的文件结构：
　　♠ Classes是项目的源文件存放位置，各种类与头文件。
　　♠ cocos2d是Cocos2d-x的各种库。
　　♠ proj.android是项目的安卓版，在项目打包成安卓安装包apk时用到。
　　♠ proj.ios_mac是项目的苹果版，使用mac本的同学利用XCode打开此文件夹中的xxx.Xcodeproj文件（xxx为项目名）既可以编辑这个项目，打包成苹果系统的包也是利用这个文件夹。
　　♠ proj.linux是项目的Linux版。
　　♠ proj.win8.1-universal应该是项目的win8.1系统版，到目前为止船长还没用过这个文件夹的文件，所以只是猜测，大家可以忽略。
　　♠ proj.win32是项目的windows系统版，使用windows系统的同学利用VS打开此文件夹中的xxx.sln文件（xxx为项目名）既可以编辑这个项目。
　　♠ proj.wp8-xaml应该是项目的WindowsPhone版，这个文件也只是猜测，大家可以忽略。
　　♠ Resources是项目的资源文件夹，项目的所有资源都放在此文件夹中。
